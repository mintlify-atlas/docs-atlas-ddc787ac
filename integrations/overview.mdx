## What are c15t integrations?

c15t provides pre-built script integrations for popular analytics and marketing tools. These integrations automatically respect user consent preferences and integrate seamlessly with c15t's script loader.

Install the scripts package alongside c15t:

<CodeGroup>
```bash npm
npm install @c15t/scripts c15t
```

```bash yarn
yarn add @c15t/scripts c15t
```

```bash pnpm
pnpm add @c15t/scripts c15t
```

```bash bun
bun add @c15t/scripts c15t
```
</CodeGroup>

## Basic usage

Import the integration function and pass it to c15t's script loader:

```typescript
import { loadScripts } from 'c15t';
import { googleTagManager } from '@c15t/scripts/google-tag-manager';

// Create the script configuration
const gtmScript = googleTagManager({
  id: 'GTM-XXXXXXX',
});

// Load with c15t's script loader
loadScripts([gtmScript], {
  consents: {
    necessary: true,
    functionality: false,
    measurement: false,
    marketing: false,
    experience: false,
  },
});
```

## How it works

### Consent categories

Each integration is assigned a default consent category:

- **necessary**: Google Tag Manager (since it manages other scripts)
- **measurement**: PostHog
- **marketing**: Meta Pixel, TikTok Pixel, LinkedIn Insights, Microsoft UET, X Pixel

You can override the category for any integration:

```typescript
const customScript = metaPixel({
  pixelId: '123456789012345',
  script: {
    category: 'measurement', // Override default 'marketing'
  },
});
```

### Script lifecycle

Integrations handle the complete script lifecycle:

1. **Before load**: Initialize vendor-specific objects and configuration
2. **On load**: Execute vendor initialization code
3. **On consent change**: Update consent state with vendor APIs
4. **On delete**: Clean up scripts and revoke consent when needed

### Persistent scripts

Some integrations (Meta Pixel, TikTok Pixel, Microsoft UET, X Pixel) are persistent — they remain loaded even when consent is revoked, but use vendor consent APIs to stop tracking. This provides better performance and user experience.
