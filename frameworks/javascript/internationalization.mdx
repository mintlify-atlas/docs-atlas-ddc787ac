# Internationalization

The c15t JavaScript SDK includes built-in internationalization (i18n) support, allowing you to display consent UI in multiple languages.

## Quick Start

Configure translations when creating the store:

```typescript
import { createConsentManagerStore, configureConsentManager } from 'c15t';
import { enTranslations, deTranslations } from '@c15t/translations';

const manager = configureConsentManager({
  backendURL: '/api/c15t',
});

const store = createConsentManagerStore(manager, {
  translationConfig: {
    translations: {
      en: enTranslations,
      de: deTranslations,
    },
    defaultLanguage: 'en',
    disableAutoLanguageSwitch: false,
  },
});
```

## Merging Translations

Merge custom translations with built-in translations:

```typescript
import { deepMergeTranslations, enTranslations } from 'c15t';

const customTranslations = {
  cookieBanner: {
    title: 'Custom Title',
    description: 'Custom description',
  },
};

const mergedTranslations = deepMergeTranslations(
  enTranslations,
  customTranslations
);

const store = createConsentManagerStore(manager, {
  translationConfig: {
    translations: {
      en: mergedTranslations,
    },
    defaultLanguage: 'en',
  },
});
```

  translationConfig: {
    translations: {
      en: enTranslations,
      de: deTranslations,
      fr: frTranslations,
    },
    defaultLanguage: browserLanguage || 'en',

});

## Automatic Language Detection

By default, c15t automatically detects the user's browser language:

```typescript
const store = createConsentManagerStore(manager, {
  translationConfig: {
    translations: {
      en: enTranslations,
      de: deTranslations,
    },
    defaultLanguage: 'en',
    disableAutoLanguageSwitch: false, // Enable auto-detection
  },
});

// c15t will use German translations if the browser is set to German
// Falls back to English if the browser language is not supported
```

## Disable Auto-Detection

Force a specific language regardless of browser settings:

```typescript
const store = createConsentManagerStore(manager, {
  translationConfig: {
    translations: {
      en: enTranslations,
      de: deTranslations,
    },
    defaultLanguage: 'en',
    disableAutoLanguageSwitch: true, // Always use default language
  },
});
```

## Example: Multi-Language E-commerce Site

```typescript
import {
  configureConsentManager,
  createConsentManagerStore,
  detectBrowserLanguage,
  deepMergeTranslations,
} from 'c15t';
import {
  enTranslations,
  deTranslations,
  frTranslations,
} from '@c15t/translations';

const manager = configureConsentManager({
  backendURL: '/api/c15t',
});

// Customize translations for your brand
const customEnTranslations = deepMergeTranslations(enTranslations, {
  cookieBanner: {
    title: 'Welcome to Our Store',
    description: 'We use cookies to personalize your shopping experience.',
  },
});

const customDeTranslations = deepMergeTranslations(deTranslations, {
  cookieBanner: {
    title: 'Willkommen in unserem Shop',
    description: 'Wir verwenden Cookies, um Ihr Einkaufserlebnis zu personalisieren.',
  },
});

const customFrTranslations = deepMergeTranslations(frTranslations, {
  cookieBanner: {
    title: 'Bienvenue dans notre boutique',
    description: 'Nous utilisons des cookies pour personnaliser votre expérience d\'achat.',
  },
});

const store = createConsentManagerStore(manager, {
  translationConfig: {
    translations: {
      en: customEnTranslations,
      de: customDeTranslations,
      fr: customFrTranslations,
    },
    defaultLanguage: detectBrowserLanguage() || 'en',
    disableAutoLanguageSwitch: false,
  },
});

// Add language switcher UI
const languageSwitcher = document.querySelector('#language-switcher');
languageSwitcher?.addEventListener('change', (e) => {
  const language = (e.target as HTMLSelectElement).value;
  const translations = {
    en: customEnTranslations,
    de: customDeTranslations,
    fr: customFrTranslations,
  };
  
  store.getState().setTranslationConfig({
    translations: { [language]: translations[language] },
    defaultLanguage: language,
    disableAutoLanguageSwitch: true,
  });
});
```
