## Overview

The core backend API provides the main entry point for creating c15t consent management instances.

## Functions

### c15tInstance

Creates a new c15t consent management instance with a unified request handler.

#### Import

```typescript
import { c15tInstance } from '@c15t/backend';
```

#### Type Signature

```typescript
function c15tInstance(options: C15TOptions): C15TInstance
```

#### Parameters

<ParamField path="options" type="C15TOptions" required>
  Configuration options for the c15t instance.
  
  <Expandable title="options properties">
    <ParamField path="adapter" type="DatabaseAdapter" required>

## Usage Examples

### Basic Setup

```typescript app/api/c15t/[...c15t]/route.ts
import { c15tInstance } from '@c15t/backend';
import { kyselyAdapter } from '@c15t/backend/adapters/kysely';
import { db } from '@/lib/db';

const instance = c15tInstance({
  adapter: kyselyAdapter(db),
  trustedOrigins: ['https://example.com'],
});

export const { GET, POST, PUT, PATCH, DELETE, OPTIONS, HEAD } = instance;
```

### With Custom Configuration

```typescript
import { c15tInstance } from '@c15t/backend';
import { drizzleAdapter } from '@c15t/backend/adapters/drizzle';
import { db } from '@/lib/db';

const instance = c15tInstance({
  adapter: drizzleAdapter(db),
  trustedOrigins: [
    'https://example.com',
    'https://app.example.com',
    'http://localhost:3000',
  ],
  appName: 'my-consent-app',
  basePath: '/api/consent',
  logger: {
    level: 'debug',
    pretty: true,
  },
  advanced: {
    branding: 'none',
    disableGeoLocation: false,
    openapi: {
      enabled: true,
      specPath: '/api/consent/spec.json',
      docsPath: '/api/consent/docs',
    },
  },
});

export const { GET, POST } = instance;
```

### Custom Request Handler

```typescript server.ts
import { c15tInstance } from '@c15t/backend';
import { kyselyAdapter } from '@c15t/backend/adapters/kysely';
import { db } from './db';

const instance = c15tInstance({
  adapter: kyselyAdapter(db),
  trustedOrigins: ['*'],
});

// Use with any server framework
const server = Bun.serve({
  port: 3000,
  async fetch(request) {
    const url = new URL(request.url);
    
    if (url.pathname.startsWith('/api/c15t')) {
      return instance.handler(request);
    }
    
    return new Response('Not Found', { status: 404 });
  },
});

console.log(`Server running at http://localhost:${server.port}`);
```

### With OpenAPI Documentation

```typescript
const instance = c15tInstance({
  adapter: kyselyAdapter(db),
  trustedOrigins: ['https://example.com'],
  advanced: {
    openapi: {
      enabled: true,
      specPath: '/spec.json',
      docsPath: '/docs',
    },
  },
});

// Access OpenAPI spec programmatically
const spec = await instance.getOpenAPISpec();
console.log(JSON.stringify(spec, null, 2));

// Get docs UI HTML
const docsHtml = instance.getDocsUI();
```

## Helper Functions

### defineConfig

Helper function for defining database configuration with TypeScript support.

```typescript
import { defineConfig } from '@c15t/backend';
import { kyselyAdapter } from '@c15t/backend/adapters/kysely';
import { db } from './db';

export const config = defineConfig({
  adapter: kyselyAdapter(db),
});
```
